# deadair.nyc

A late-night listening bar. A dark, grainy, CRT-glow web player for shoegaze /
alt / 90s–2000s / indie. The conceptual hook: **dead air is the feature** — when
nothing's playing, the player is tuned to procedurally-generated static, like an
off-air channel. You _power on_ to tune in.

Built with **SvelteKit (Svelte 5 runes) + TypeScript**, fully static (no backend).

## Develop

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # static output → build/
npm run preview    # serve the production build
npm run check      # svelte-check (types)
```

## Add your own music

This is the whole job:

1. Drop an audio file in `static/audio/` (mp3, wav, ogg…).
2. Add an entry to `src/lib/data/tracks.ts`:

   ```ts
   { id: '04', title: 'Your Song', artist: 'You', src: 'audio/your-song.mp3' }
   ```

No other code changes needed — the tracklist, player, and visualizer read from
that manifest.

> The bundled `lowlight/glass/bloom.wav` are **original, royalty-free** ambient
> drones generated by `scripts/gen-audio.mjs` — placeholders to prove the pipeline.
> Delete them and the matching entries in `tracks.ts` when you add real tracks.

## How it works

- `src/lib/audio/engine.ts` — one `AudioContext`. Signal chain:
  `noise + <audio>` → master gain → **analyser** → speakers. The shared analyser
  feeds the on-screen visualizer, so the screen reacts to whatever is playing.
- `src/lib/audio/noise.ts` — procedural pink/white noise (the dead air).
- `src/lib/stores/player.svelte.ts` — runes store; the single source of truth the
  UI reads. Methods drive the engine; the engine reports time/ended back.
- `src/lib/components/` — `Desktop` (background + clock + grain/scanlines),
  `PlayerWindow` → `Screen` (`Visualizer`) + `NowPlaying` + `Tracklist` + `Transport`.

**Autoplay note:** browsers block audio until a user gesture, so the first click is
the **POWER ON** button — it resumes the `AudioContext` and starts the static.

## Deploy

The build is fully static (`@sveltejs/adapter-static`). Point any static host at
`build/`. Cloudflare Pages (free) is a good fit — connect the repo, build command
`npm run build`, output dir `build`, then attach the `deadair.nyc` domain.
